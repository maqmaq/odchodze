# Odchodze

## How to run it localy

Run command:
```bash
php phing-latest.phar docker:up
```

Wait for composer installation to finish.
You can check progress using:
```bash
docker logs -f odchodze-php-fpm
```

Visit:

[http://localhost:8080](http://localhost:8080)
